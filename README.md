# Samiul Hossen Sarkar Santo - Portfolio Website

Personal portfolio website of **Samiul Hossen Sarkar Santo** showcasing AI/ML, web, and mobile projects, along with experience, education, and contact information.

## Overview

This is a static portfolio site built from the **Luther** template and customized with personal branding, project content, and updated project links.

Sections included:
- Intro
- About
- Technical Skills
- Experience
- Education
- Selected Projects (with modal previews)
- Contact

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Third-party frontend libraries bundled in template assets

## Run Locally

1. Clone the repository.
2. Open the `public` folder.
3. Open `index.html` directly in a browser, or use a local static server (recommended for development).

Example with VS Code Live Server:
- Right-click `index.html`
- Click `Open with Live Server`
